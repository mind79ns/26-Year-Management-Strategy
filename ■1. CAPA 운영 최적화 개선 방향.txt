■1. CAPA 운영 최적화 개선 방향
  - 1일 재고 CAPA 기준 확인 (6만~7만) 
  - SMD 공정 LINE 운영 최적화 검토 ( 현 운영라인 5개 기준 OVERTIME 운영 비용과 인원TO 추가 7EA LINE 운영 OVERTIME 감소운영 비용 차이점 비교)
  - 인원 추가 대비 최소인력 운영비용 외 전력소모 감소예상 포함 분석 
  - OVERTIME 만 감안하고 인원 추가 및 LINE CAPA 늘렸을때 1일 재고가 OVER 되는 경우는 불필요 낭비요소 발생되는지 검토 필요
  - OVER TIME 에 발생되는 비용 산출 과 1명 인원 늘어나는 인건비용(보험+기타+급여 등등) 분석필요
  - 모든 결과값을 기준으로 26년 LINE 운영 최적화 방향 과제 제시

■ 2. 설비 CAPA 향상 과제
  - 각 LINE 의 CAPA 향상을 하기위한 과제활동 내용 정리 필요함.
  - 설비의 PM 및 성능유지 등 기본 관리 BASE로 그동안 전략 해온 모든것들은 지속 관리 운영한다
  - 기회 손실 비용을 최소화 하기위한 효율상승 목적 → 순간유실 관련 DATA DEEP 하게 접근 분석 및 자동 TOOL 개발 스마트 현장 운영 과제
  - C/T 최소화 할수 있는 최단경로 DATA 검증 방법
  - SMD LINE 설비 LAY OUT 변경 주력 LINE CAPA 최적화 퍼포먼스 구현 계획 활동 ( 예시 주 생산 모델 1시간 2000개 → 2100개 설비 마운터 재배열 )
